# 栈的特性

# 1. 操作受限的线性表

  • 先进后出(First in, last out)

# 2. 两个基本操作

  • 入栈(push)
  • 出栈(pop)

# 栈的实现

  • 顺序栈(使用数组实现)
    • 支持动态扩容的顺序栈: 与数组动态扩容类似
  • 链式栈(使用链表实现)
    • 天然支持动态扩容，但是需要更多的内存空间(储存next指针)

# 栈的常见应用

  • 函数调用，栈帧
  • 表达式求值(使用两个栈，分别存数值和运算符)
  • 括号匹配
  • 浏览器前进后退(使用两个栈)
