# 回溯算法、贪心算法、动态规划比较

  • 贪心算法：一条路走到黑，就一次机会，只能哪边看着顺眼走哪边
  • 回溯算法：一条路走到黑，无数次重来的机会，还怕我走不出来 (Snapshot View)
  • 动态规划：拥有上帝视角，手握无数平行宇宙的历史存档， 同时发展出无数个未来 (Versioned Archive View)
