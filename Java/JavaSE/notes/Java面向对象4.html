# Java抽象类

面向对象概念中，对象都是通过类来描述，但并不是所有类都是用来描绘对象的

包含抽象方法的普通类->抽象类

对子类有所要求: 可以强制子类对某些方法进行实现(覆写)

# 特性

  • 抽象类不能被实例化对象，必须被继承才能使用
  • 抽象类的子类需要实现所有的抽象方法，否则也是抽象类
  • 构造方法，类方法(静态方法)不能声明为抽象方法

# 注意

  • 抽象类允许不包含抽象方法，但依然不能被实例化
  • 子类对抽象方法的实现也算覆写 (@Override不会报错)
  • 日常开发通常不会继承普通类，只继承抽象类
