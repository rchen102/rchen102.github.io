## 树
| 操作 |  平均时间  | 最差时间（树退化为链表） |
| :--: | :--: | :--: |
| 查找 | O(logn) | O(n) | 
| 插入 | O(logn) | O(n) | 
| 删除 | O(logn) | O(n) |  
| 遍历 | O(n)    | O(n) |

## 堆
| 操作 |  平均时间  |
| :--: | :--: |
| 建堆| O(n) |
| 弹出堆顶 | O(logn) |  
| 插入元素 | O(logn) | 
| 删除任意 | O(logn) |
