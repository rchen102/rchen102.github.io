## 介绍
哈希算法：一种映射规则（函数），能够将任意长度的二进制值串映射为**固定长度**的二进制值串

## 目标
- 从哈希值不能反向推导原数据
- 对输入数据敏感，哪怕只修改了 1 个 bit，哈希值也会不同
- 散列冲突的概率要尽可能小
- 执行效率尽可能高效

## 应用
1. 作为唯一标识，对大数据做信息摘要，通过一个较短的二进制编码来代表一个很大的数据
2. 校验数据的完整性和正确性
3. 安全加密
4. 作为散列函数使用
5. 负载均衡
6. 数据分片
7. 分布式存储
