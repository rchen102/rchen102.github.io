# 服务启动

  • 通过服务窗口 services.msc
  • 以管理员模式运行 net stop/start mysql

# 登录/退出

  • mysql -uroot -proot 明文显示用户名和密码
  • mysql -uroot -p 隐藏密码显示
  • mysql -hip -uroot -proot 远程连接登录
  • exit或quit 退出

# 目录结构

  • MySQL安装目录: 配置文件my.ini
  • MySQL数据目录:
    • 数据库: 文件夹
    • 表: 文件
    • 数据: .frm
